根据百度百科上说的，Eclipse 是一个开放源代码的、基于Java的可扩展开发平台。就其本身而言，它只是一个框架和一组服务，用于通过插件组件构建开发环境。幸运的是，Eclipse 附带了一个标准的插件集，包括Java开发工具（Java Development Kit，JDK）。

但是对我们大部分的开发者来说是将 Eclipse 当作 Java 集成开发环境（IDE）来使用了。本文将介绍一些eclipse的插件和快捷键使用。
