springboot集成dubbo是近来比较流行的一种组合，博主闲来无聊的时候也试了一下，发现整合其实还算是很容易的，但是如何更好的使用是个问题。

在分布式环境有就有生产者和消费者的区分了。我们首先设计接口，以供消费者和生产者使用。这样做有一个好处就是接口与实现隔离。

这里演示的接口包很简单，包含一个实体和一个service接口。

实体：

                  
  public class User implements Serializable{

	private static final long serialVersionUID = 1L;
	
	private String name;
	private int age;
	public String getName() {
		return name;
	}
	public void setName(String name) {
		this.name = name;
	}
	public int getAge() {
		return age;
	}
	public void setAge(int age) {
		this.age = age;
	}

 }

接口：

                  
   public interface UserService {
	
		public User getUser();
   }
